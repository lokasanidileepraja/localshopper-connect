## Component Specifications

### Button Variants

**Primary Button**
- Background: Primary gradient
- Text: White, 600 weight
- Padding: 16px 32px
- Border Radius: 9999px (pill)
- Shadow: 0 4px 14px rgba(102,126,234,0.4)
- Hover: translateY(-2px), shadow increase

**Secondary Button**
- Background: Secondary
- Text: Foreground
- Border: 1px solid Border
- Same padding/radius

**Ghost Button**
- Background: Transparent
- Text: Muted Foreground
- Hover: Background secondary

**Outline Button**
- Background: Transparent
- Border: 2px solid Primary
- Text: Primary

### Card Variants

**Product Card**
- Background: Card
- Border: 1px solid Border
- Border Radius: 16px
- Shadow: Shadow MD
- Padding: 16px
- Hover: translateY(-8px), Shadow XL

**Glass Card**
- Background: rgba(255,255,255,0.08)
- Backdrop Filter: blur(24px)
- Border: 1px solid rgba(255,255,255,0.12)
- Shadow: Glass Shadow

**Store Card**
- Similar to Product Card
- Includes: Image, name, distance, rating, status badge

### Input Fields

**Default Input**
- Height: 44px
- Border: 1px solid Border
- Border Radius: 8px
- Padding: 0 16px
- Focus: Ring 2px Primary

**Search Input**
- Has search icon (left)
- Clear button (right, on content)
- Larger: 52px height for hero

### Badges

**Status Badges**
- In Stock: Green background, green text
- Out of Stock: Red background
- Verified: Blue with checkmark icon
- Sale: Primary gradient
